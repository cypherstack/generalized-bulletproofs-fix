# generalized-bulletproofs-fix

## Notes on Bulletproof Generalizations

This repository contains a single PDF with a draft technical report intended for Monero Research Lab, and related to the correction of certain incorrect generalizations of Bulletproofs-style proving systems.

These changes are important to security in upcoming full-chain membership proofs for Monero, but the report rehashes older information, and is likely not suitable for peer-reviewed publication.

The document is a work in progress and is being shared primarily for limited review and reference. A final, complete version which will be ready for citation is forthcoming.
